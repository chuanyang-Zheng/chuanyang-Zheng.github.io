COMP4015 Artificial Intelligence And Machine Learning Course Report
– CycleGAN only cycles once. According to Squeeze Theorem, we add additional cycles by regards cycle-consistent images as real images. There are two advantages compared to the original CycleGAN. First, according to Squeeze Theorem, the cycle-consistent should be much closer to the original real images. Second, we can get more images as input because we regard cycle-consistent images as real images
